Dota Helper

Open-source app for the game Dota 2 that puts an overlay with some information.

GitHub

FEATURES

GPM & XPM INFO

In the upper left corner information about the gold per minute and experience per minute is shown.
Hold 'ALT' to show info about gold.

FULLSCREEN

AEGIS TIMER

Under hero portrait shows live of aegis.
(Unfortunately, game doesn't send info when someone dies with aegis. To hide timer use 'ALT+F1')

FULLSCREEN

ABILITIES COOLDOWN

Now you can see which ability in cooldown on invoker and kez (with scepter).

FULLSCREEN

CONJURE IMAGE SPAWN PREDICT

Did you know that terrorblade's second ability has spawn pattern? Overlay will show you in which next way illusion will spawn.

FULLSCREEN

REQUIREMENTS

Before starting Dota 2 and Dota Helper, make sure that you have the special config installed.
1. Download gamestate_integration_dotahelper.cfg.
2. Go to folder where Dota 2 installed.
3. Then to this path \game\dota\cfg\gamestate_integration.
4. Put downloaded file to that folder.
5. Done!
